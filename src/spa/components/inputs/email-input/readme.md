### Event Listeners

The `input` event will fire while the user types.

```typescript
document.body.querySelector('brixi-email-input').addEventListener('input', (e) => {
    const { name, value } = e.detail;
});
```

The `focus` event will fire when the user focuses the input.

```typescript
document.body.querySelector('brixi-email-input').addEventListener('focus', (e) => {
    const { name, value } = e.detail;
});
```

The `blur` event will fire when the user blurs the input.

```typescript
document.body.querySelector('brixi-email-input').addEventListener('blur', (e) => {
    const { name, value } = e.detail;
});
```
