### Types

```typescript
type ItemStyle = "disc" | "circle" | "decimal" | "leading-zero" | "square" | "custom";
type ListType = "ordered" | "unordered";
type List = {
    type: ListType;
    style?: ItemStyle;
    items: Array<string>;
    sub?: List;
    icon?: string;
};
```
