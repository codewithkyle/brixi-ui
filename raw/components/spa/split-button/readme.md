Not sure what Data Attributes are? Learn about [Data Attributes on MDN](https://developer.mozilla.org/en-US/docs/Web/HTML/Global_attributes/data-*).

### Types

```typescript
type OverflowItem = {
    label: string;
    id: string;
    icon?: string;
    danger?: boolean;
};
type ButtonType = "submit" | "button" | "reset";
```
